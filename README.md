# Pantone Solid Coated Quick Reference

ブラウザで Pantone Solid Coated 約3,219色を素早く参照するためのリファレンスサイト。

🌐 **公開URL**: https://fujimoto-cpu.github.io/pantone/

## 機能

- 色相順 / 番号順 / 明度順で並び替え
- 番号・名称・HEX で検索
- CMYK 値から最近傍 Pantone を検索（ΔE2000 距離・上位5件）
- 各色クリックで HEX / CMYK / Lab 値の表示・コピー

## データ出典

色データは [aj90909/unofficial-pantone-solid-coated-2024-v5](https://github.com/aj90909/unofficial-pantone-solid-coated-2024-v5) を採用。
Photoshop で抽出された非公式の Lab 値であり、印刷現場での厳密な指定には公式 Pantone カラーブックを参照。
